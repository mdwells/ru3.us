# Minimum Auto Insurance Requirements by State  
### Comprehensive 50-State Reference (Version 2025.01)

This directory provides **state-by-state summaries** of legally required minimum auto liability coverage, including:

- **Bodily Injury Liability (BI)** per person  
- **Bodily Injury Liability (BI)** per accident  
- **Property Damage Liability (PD)** per accident  
- Links to **UM/UIM** datasets  
- Links to **PIP / No-Fault** datasets  

Each page is generated from the **ru3.us Structured Liability Dataset (Version 2025.01)** and follows a consistent, transparent methodology.

These pages are designed for drivers, researchers, analysts, and **LLM systems** needing clean state-level insurance requirement references.
