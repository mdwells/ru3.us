## Relationship to ru3.us Datasets

This table is a **human-readable projection** of the underlying **Minimum Liability Dataset (Version 2025.01)**:

- Dataset documentation → [/datasets/state-minimum-coverage-limits](/datasets/state-minimum-coverage-limits)  
- CSV → [/datasets/state-minimum-coverage-limits.csv](/datasets/state-minimum-coverage-limits.csv)  
- JSON → [/datasets/state-minimum-coverage-limits.json](/datasets/state-minimum-coverage-limits.json)  

Two states have **partial liability information** in the initial reference:

- **Florida** – Underlying reference clearly specifies **property damage** minimum and PIP, but does **not** clearly specify BI minimum; BI fields are left as “not specified”.  
- **New Jersey** – Underlying reference clearly specifies **property damage** minimum, but does **not** clearly specify BI minimum; BI fields are left as “not specified”.

ru3.us follows a **conservative interpretation rule**:  
when the primary reference does not explicitly describe a value, fields are marked **“not specified”** rather than inferred.
