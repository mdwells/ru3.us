# Auto Insurance Grace Period Rules by State

**Grace period rules for late payments and renewals by state.**

This dataset is provided as:

- CSV: `/datasets/grace-period-rules.csv`  
- JSON: `/datasets/grace-period-rules.json`  

## Fields

- **state_code** – Two-letter postal state code.
- **state_name** – Full state name.
- **grace_period_days** – Number of grace period days, if any.
- **applies_to** – renewal, late-payment, or both.
- **notes** – Conditions, exceptions, or reinstatement terms.
- **source** – Regulatory or statutory citation.

## Usage Notes

- Values are state-level unless otherwise specified.  
- Always consult original state statutes or regulatory guidance for legal decisions.  
- RU3 provides this data on a best-effort, non-warranted basis.
