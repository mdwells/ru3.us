# Uninsured and Underinsured Motorist Coverage Requirements by State

**Requirements for UM and UIM coverage by state.**

This dataset is provided as:

- CSV: `/datasets/um-uim-requirements.csv`  
- JSON: `/datasets/um-uim-requirements.json`  

## Fields

- **state_code** – Two-letter postal state code.
- **state_name** – Full state name.
- **um_required** – Is uninsured motorist coverage required? (yes/no/offer-only).
- **uim_required** – Is underinsured motorist coverage required? (yes/no/offer-only).
- **combined_or_separate** – Whether UM/UIM limits are combined or separate.
- **notes** – Additional requirement details or exceptions.

## Usage Notes

- Values are state-level unless otherwise specified.  
- Always consult original state statutes or regulatory guidance for legal decisions.  
- RU3 provides this data on a best-effort, non-warranted basis.
