# No-Fault vs. Tort Auto Insurance Systems by State

**Classification of each state as no-fault, tort, or choice for auto insurance.**

This dataset is provided as:

- CSV: `/datasets/no-fault-vs-tort.csv`  
- JSON: `/datasets/no-fault-vs-tort.json`  

## Fields

- **state_code** – Two-letter postal state code.
- **state_name** – Full state name.
- **system_type** – no-fault, tort, or choice.
- **notes** – Thresholds or conditions in no-fault states.

## Usage Notes

- Values are state-level unless otherwise specified.  
- Always consult original state statutes or regulatory guidance for legal decisions.  
- RU3 provides this data on a best-effort, non-warranted basis.
