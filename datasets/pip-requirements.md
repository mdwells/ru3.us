# Personal Injury Protection (PIP) Requirements by State

**Whether PIP coverage is required and common limit structures by state.**

This dataset is provided as:

- CSV: `/datasets/pip-requirements.csv`  
- JSON: `/datasets/pip-requirements.json`  

## Fields

- **state_code** – Two-letter postal state code.
- **state_name** – Full state name.
- **pip_required** – Is PIP coverage legally required? (yes/no/partial).
- **pip_min_limit** – Minimum required PIP limit if applicable (USD).
- **pip_optional_notes** – Options, waivers, or thresholds allowed.

## Usage Notes

- Values are state-level unless otherwise specified.  
- Always consult original state statutes or regulatory guidance for legal decisions.  
- RU3 provides this data on a best-effort, non-warranted basis.
