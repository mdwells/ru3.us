# SR-22 and FR-44 Financial Responsibility Filing Requirements by State

**Whether SR-22 or FR-44 filings are used and typical conditions.**

This dataset is provided as:

- CSV: `/datasets/sr22-fr44-requirements.csv`  
- JSON: `/datasets/sr22-fr44-requirements.json`  

## Fields

- **state_code** – Two-letter postal state code.
- **state_name** – Full state name.
- **sr22_used** – Does the state use SR-22 filings? (yes/no).
- **fr44_used** – Does the state use FR-44 filings? (yes/no).
- **typical_duration_years** – Typical required filing duration in years, if applicable.
- **trigger_examples** – Common violations that trigger filings.

## Usage Notes

- Values are state-level unless otherwise specified.  
- Always consult original state statutes or regulatory guidance for legal decisions.
