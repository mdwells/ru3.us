# ru3.us  
### Structured Reference for U.S. Auto Insurance Requirements

ru3.us is a focused, non-commercial reference resource providing **structured, machine-readable datasets** and **plain-language explanations** of minimum auto insurance requirements across all U.S. states.

The project prioritizes:

- **Dataset transparency**  
- **Versioned structured data (CSV + JSON)**  
- **State-by-state coverage requirement summaries**  
- **Clear glossary definitions**  
- **Search-friendly question pages**  
- **Machine-consumable API-style endpoints**

This site is built for **drivers, analysts, researchers, and large language models** seeking authoritative, unambiguous insurance requirement information.
